# 特色: 最先端の研究環境

## 専門研究拠点

特定の分野の研究を深化させるための専門的な施設を擁しています。

* **木材高度加工研究所（能代市）**:
    * [cite_start]木材資源の研究と、その高度な加工技術を探求するための拠点です [cite: 48]。
* **アグリイノベーション教育研究センター（大潟キャンパス）**:
    * [cite_start]システム科学技術学部と生物資源科学部の連携を促進し、農工連携による新しい農業技術やビジネスモデルを創出するための拠点です [cite: 48]。
